## Installation

色々なプロジェクトで使う画像エンコーダ・デコーダの実装をまとめたパッケージ

### pip

```shell
pip install git+https://github.com/nomutin/CNN.git
```

### poetry

```shell
poetry add git+https://github.com/nomutin/CNN.git
```

### rye

```shell
rye add cnn --git=https://github.com/nomutin/CNN.git
```
